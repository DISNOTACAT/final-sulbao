## 프로젝트 소개
- 항공사 직원이 항공권을 발급하고 발급정보를 수정하거나 검색하는 시스템입니다.
- 출발지는 한국으로 고정되어있습니다.
- 이름은 영문으로만 입력 가능하며, 도착지는 한국어로만 입력이 가능합니다.
- 항공권 번호는 도착지 영문3자리와 랜덤숫자 3자로 구성되어있습니다.
